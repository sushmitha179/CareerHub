# CareerHub

CareerHub is a full-stack placement portal built for students and recruiters. The platform allows recruiters to post jobs and students to apply and track applications through personalized dashboards.

## Features

- User Authentication
- Student Dashboard
- Recruiter Dashboard
- Job Posting
- Job Listings
- Application Tracking
- Profile Management
- Resume Upload
- AI Resume Analyzer (In Progress)

## Tech Stack

- Next.js
- Prisma ORM
- PostgreSQL
- Tailwind CSS
- TypeScript

## Installation

Clone the repository:

```bash
git clone https://github.com/sushmitha179/CareerHub.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
